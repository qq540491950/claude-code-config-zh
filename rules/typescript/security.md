# TypeScript 安全规范

> 此文件扩展 [common/security.md](../common/security.md) 添加 TypeScript 特定内容。

## 输入验证

- 所有外部输入（请求参数、表单、URL 参数）必须验证
- 推荐使用基于模式的验证（如 Zod）
- 校验失败应快速返回并提供可理解错误

## 前端安全

- 禁止渲染未经清洗的 HTML
- 严禁把敏感令牌写入可公开日志
- 处理用户输入时防止 XSS 注入

## 服务端安全（Node/TS）

- 数据库访问使用参数化查询
- 鉴权与授权必须在边界层校验
- 错误响应不得暴露堆栈与敏感配置

## 密钥管理

- 禁止硬编码 API Key、Token、密码
- 使用环境变量并在启动时校验必需项
