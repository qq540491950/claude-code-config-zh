# JavaScript 安全规范

> 此文件扩展 [common/security.md](../common/security.md) 添加 JavaScript/Node 特定内容。

## 输入校验

- 所有外部输入（请求体、查询参数、headers）必须校验
- 校验失败返回明确错误，不进入业务逻辑

## 服务端安全

- 数据库访问必须使用参数化查询
- 错误响应不得泄露堆栈和敏感配置
- 鉴权与授权应在路由边界执行

## 前端安全

- 禁止直接渲染未经清洗的用户输入 HTML
- 对 URL 参数和动态路由输入做白名单校验

## 密钥与配置

- 禁止硬编码 token、密钥、数据库连接串
- 通过环境变量注入并在启动阶段做必需项校验
